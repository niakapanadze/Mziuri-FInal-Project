# 🚀 Space Project

A web-based Space Management System built with Flask that allows an admin to manage planets. The admin can add, update, and delete planets using a secure login system.

---

## 🌌 Features

- Admin authentication (Flask-Login)
- Add new planets
- Update planet details
- Delete planets
- View list of all planets
- Form handling with Flask-WTF
- Database integration using Flask-SQLAlchemy
- Simple UI with HTML and CSS

---

## 🛠️ Tech Stack

- Python
- Flask
- Flask-WTF
- Flask-SQLAlchemy
- Flask-Login
- HTML
- CSS
